# Генерация презентации

Быстрая инструкция по созданию PDF-презентации компании.

## Команды

```bash
# 1. Экспорт данных из CMS
pnpm export:homepage
pnpm export:cases

# 2. Подготовка сокращенных текстов для слайдов
pnpm prepare:cases

# 3. Генерация HTML презентации
pnpm generate:presentation

# 4. Открыть в браузере
open presentation.html
```

## Сохранение в PDF

1. Откройте `presentation.html` в браузере
2. Нажмите **Cmd+P** (или Ctrl+P на Windows)
3. Настройки:
   - **Принтер**: Сохранить как PDF
   - **Ориентация**: Альбомная
   - **Размер**: A4
   - **Поля**: Без полей
   - **Фон**: ✅ Включить

## Структура презентации

- Титульный слайд (лого + заголовок + контакты)
- Целевая аудитория (4 сегмента)
- Кейсы: по 2 слайда на каждый (описание + скриншоты)
- Услуги (3 этапа)
- Итоговый слайд (призыв к действию)
